# GymITS
___
## How To Run
- `npm install`
- `npm run start`
- For new database creation
- `npm run up`
- `npm run seed`
- To reset database
- `npm run down`

## When Editting FrontEnd
- `npm run tailwind`

## Information
- All FrontEnd works will be put on /public, so just modify the stuff in there, but don't modify elsewhere
